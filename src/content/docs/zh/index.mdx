Jiang 是一门正在开发中的系统编程语言。当前重点是稳定清晰的核心语言、直接的内存与布局控制能力，以及逐步走向自举编译器。

这份文档优先作为教程和使用指南来写。它会说明当前已经形成的语言规则，也会在仍处于 Stage1 迁移中的部分保持谨慎。

## 项目状态

Jiang 正在从 Stage0 原型编译器过渡到 Stage1 自举编译器。Stage0 使用 C 实现，用来验证语言行为；Stage1 使用 Jiang 编写，目标是成为长期维护的编译器。

现阶段可以把官网文档视为持续更新的语言指南，而不是完全冻结的规范。相对稳定的内容包括命名风格、类型写法、分层可变性、数组、指针、可选类型、导入规则和当前编译器命令行工作流。
