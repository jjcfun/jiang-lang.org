Jiang 是一门正在开发中的系统编程语言。当前重点是稳定清晰的核心语言、直接的内存与布局控制能力，以及已经可以通过 0.2.0 macOS arm64 release 安装的自举编译器。

这份文档优先作为教程和使用指南来写。它会说明当前已经形成的语言规则，也会在仍处于快速演进中的部分保持谨慎。

Jiang 目前仍处于 0.2 版本阶段，现阶段看上去或许平平无奇。这里先卖个关子：0.4 版本会引入一个
杀手级特性，它会是这门语言真正拉开差异的起点。

## 项目状态

Jiang 语言编译器已经完成初步稳定自举：release 编译器由 Jiang Stage2 代码库生成，并通过 smoke
和语言测试验证。接下来的重点是扩充语言测试、完善 release 打包体验、改进 package/模块易用性，
以及继续补齐后端和运行时能力。

现阶段可以把官网文档视为持续更新的语言指南，而不是完全冻结的规范。相对稳定的内容包括命名风格、
类型写法、分层可变性、数组、指针、可选类型、导入规则、package 规则和当前编译器命令行工作流。
