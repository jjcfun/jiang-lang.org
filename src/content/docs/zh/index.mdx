Jiang 是一门正在开发中的系统编程语言。自定义语法是 Jiang 实现 All in one 的基石：
用户可以基于自己的理想语法扩展语言，同时继续享受 Jiang 自带的类型检查和系统级运行效率。
当前编译器已经可以通过 0.4.2 macOS arm64 发布包安装。

这份文档优先作为教程和使用指南来写。它会说明当前已经形成的语言规则，也会在仍处于快速演进中的部分保持谨慎。

Jiang 0.4.2 延续 0.4 系列，补齐包依赖、语言包自定义语法、拥有型动态 trait 值、
发布模式后端路径，以及更完整的源码级语言测试。

## 项目状态

Jiang 语言编译器已经完成初步稳定自举：发布版编译器由 Jiang 源码生成，并通过冒烟测试和语言测试验证。
接下来的重点是扩充语言测试、完善发布包体验、改进包和模块的易用性，以及继续补齐后端和运行时能力。

现阶段可以把官网文档视为持续更新的语言指南，而不是完全冻结的规范。相对稳定的内容包括命名风格、
类型写法、分层可变性、数组、指针、可选类型、导入规则、包规则和当前编译器命令行工作流。
