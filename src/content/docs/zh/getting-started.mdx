本指南假设 `jiangc` 已经安装好，并且可以从 `PATH` 中直接运行。如果还没有，请先阅读[安装](/zh/install/)。

## 编写程序

创建 `hello.jiang`：

```c
Int add(Int a, Int b) {
    return a + b;
}

Int main() {
    return add(40, 2);
}
```

Jiang 函数一定有返回类型。程序入口是 `main`；这里返回 `42`，也就是宿主进程的退出码。

## 编译为可执行文件

```bash
jiangc hello.jiang -o hello
```

运行程序：

```bash
./hello
echo $?
```

结果应该是 `42`。

## 查看 LLVM IR

语言开发阶段，经常需要查看生成的 LLVM IR：

```bash
jiangc --emit-llvm hello.jiang
```

使用 `-o` 可以写入文件：

```bash
jiangc --emit-llvm hello.jiang -o hello.ll
```

## Jiang 代码的基本形态

Jiang 源文件以声明为核心。一个小文件通常包含类型声明、函数和 `main` 函数：

```c
struct Pair {
    Int left;
    Int right;
}

Int sum_pair(Pair pair) {
    return pair.left + pair.right;
}

Int main() {
    Pair pair = Pair(left: 20, right: 22);
    return sum_pair(pair);
}
```
