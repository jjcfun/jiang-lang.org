可选类型是语言内建类型层，使用 `T?` 表示。它不是普通泛型类型。

## 基本用法

```c
Int? a1 = 123;
_ a2 = a1;           // a2 为 Int? 类型

Bar? b1 = Bar(x: 1, y: 2)
_ b2 = b1?.x;        // b2 为 Int? 类型
```

## 可选链

通过 `?.` 安全访问嵌套字段：

```c
_ foo = x?.y?.z;
```

## 条件解包

使用 `is some` 模式匹配解包可选值：

```c
if a1 is some x {
  // x 类型为 Int，值不为 null
} else {
  // x 为 null
}

// 显式写出推导类型
if a1 is some _ x {
  // 等价于 some x
}

// 可变绑定
if a1 is some _! x {
  // x 为可变绑定
}
```

## `??`

`??` 在可选值为空时提供备用值：

```c
Int value = maybe ?? 42;
Int other = maybe ?? default_value();
```

规则：

- 左侧必须是可选值。
- 右侧是备用值，类型要能与解包后的成功值统一。
- `??` 不用于 `return`、`break` 或 `continue` 这类提前退出。

## 提前退出：`guard`

需要“为空就提前退出”时使用 `guard`：

```c
() print_value(Int? maybe) {
    guard maybe is some value else {
        return ();
    }

    print(value);
    return ();
}
```

`guard` 的 `else` 代码块必须非空，最后一条语句必须是 `return`、`break`、`continue` 或 `throw`。

## 强制解包

`maybe$.some()` 用于强制解包可选值。它属于隐式操作层；普通代码优先使用 `is some`、`guard` 或 `??`。

```c
Int value = maybe$.some();
```
