Jiang 数组是固定长度数组，长度写在类型里，不能在运行时改变。

## 不可变数组

```c
Int[3] a = [1, 2, 3];

// 长度不匹配会导致编译错误
// Int[5] a = [1, 2, 3];  // 错误

// 在堆上创建拥有所有权的指针
Int[3]^ x = new [1, 2, 3];

// 成员访问会自动穿过拥有所有权的指针
print("array %d", x.length);
```

## 类型推断

使用 `_` 或 `Int[_]` 可以从初始化器推断数组长度：

```c
// 以下三种方式等价
_ a = [1, 2, 3];
Int[_] b = [1, 2, 3];
Int[3] c = [1, 2, 3];
```

## 数组可变性

数组长度固定，但元素可变性和绑定可变性可以独立控制：

```c
// 元素可变
Int![_] b = [1, 2, 3];
b[1] = 4;  // b: [1, 4, 3]

// 绑定可变，元素不可变
Int[_]! c = [1, 2, 3];
c = [4, 5, 6];

// 元素和绑定都可变
Int![_]! d = [1, 2, 3];
d[1] = 4;
d = [4, 5, 6];

// 拥有所有权的指针绑定本身可变
Int[_]^! e = new [1, 2, 3];
e = new [4, 5, 6];
```

## 多维数组

Jiang 没有真正的多维数组——多维数组就是嵌套数组。以 `Int[2][3]` 为例：从里往外看，最外层有 3 个元素，每个都是 `Int[2]` 类型。

```c
Int[2][3] foo = [[1, 2], [3, 4], [5, 6]];

foo[0];      // [1, 2]
foo[0][1];   // 2
```

类型的阅读顺序是 **从左往右，从里到外**：`Int -> Int[2] -> Int[2][3]`，类型范围逐渐扩大。简单来说：**越靠右边，范围越大**。

指针类型也遵循同样规则：

```c
Int?[2][3] b = [[1, null], [3, 4], [5, 6]];
Int?[2]?[3] c = [[1, null], null, [5, 6]];

Int[3]^ owner = new [1, 2, 3];
Int?[3]^ maybe_items = new [1, null, 3];
```

## 切片

`T[]` 是 `Slice<T>` 的短写，表示不定长数组指向类型。需要一段连续元素的运行时长度借用视图时，
写成 `T[]&`，也就是 `Reference<Slice<T>>`。
借用切片可以读取 `length`，但不拥有底层存储。

```c
Int[_] x = [1, 2, 3];
Int[]& y = x[..];

print("y.length = %d", y.length);  // 输出：y.length = 3
```

哨兵切片写成 `T[:S]&`，也就是 `Reference<SentinelSlice<T, S>>`。字符串字面量的自然类型是 `UInt8[:0]&`；如果非字面量
哨兵切片需要作为普通 `UInt8[]&` 使用，需要显式写 `text.slice()` 或
`text$.as(UInt8[]&)`：

```c
UInt8[:0]& text = "hello";
UInt8[]& bytes = text.slice();
```

切片下标访问返回元素引用：

```c
_ item_ref = y[1]$.ref();  // item_ref: Int&
```

`T[]&` 不拥有底层内存。把切片保存到结构体字段或返回给调用方时，需要保证被引用的数组或缓冲区生命周期覆盖切片的使用范围。

标准库 `Vector<T>` 提供两种常用转换：`vector.slice()` 返回借用视图 `T[]&`；
`vector.into_array()` 会消耗整个 `Vector`，并把已初始化元素转移给返回的 `T[]^` 拥有。
调用 `into_array()` 后，原 `Vector` 已经失效，不能继续使用。

## 多元素裸指针 `T[*]`

`T[*]` 是裸指针的一种，表示可按元素下标访问的一段连续地址。它没有运行时长度，也不是普通可遍历值；
通常要和长度值或切片一起使用。

```c
UInt8[*] data;
Int length;

UInt8 first = data[0];
```

`T*` 是单对象裸指针，不用于自由位置移动；需要按位置访问连续内存时使用 `T[*]`。
`T[*]` 不默认自动解引用，只能通过下标访问元素。需要表达运行时长度借用视图时，优先使用 `T[]&`。
