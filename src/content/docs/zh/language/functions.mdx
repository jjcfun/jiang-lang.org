函数声明先写返回类型，然后是函数名、参数和函数体。

```c
Int add(Int left, Int right) {
    return left + right;
}
```

## 返回类型

函数总是有返回类型。没有有意义的返回值时使用 `()`表示Unit类型：

```c
() log_done() {
    print("done");
    return ();
}

```

## 参数

参数按位置匹配：

```c
Int scale(Int value, Int factor) {
    return value * factor;
}

Int result = scale(21, 2);
```

Stage1 目标语言不依赖参数标签和默认参数。旧的 Stage0 示例中可能还能看到相关语法，但新代码建议使用简单的位置参数调用。

## 结构体值

`struct` 是带命名字段的名义类型：

```c
struct Pair {
    Int left;
    Int right;
}

Int sum_pair(Pair pair) {
    return pair.left + pair.right;
}

Int main() {
    Pair pair = Pair { left: 20, right: 22 };
    return sum_pair(pair);
}
```

## 多返回值

可以使用 tuple 返回多个值：

```c
(Int, Int) split(Int value) {
    return (value, value + 1);
}

Int main() {
    (_ left, _ right) = split(41);
    return left + right;
}
```

`(T)` 与 `T` 等价；一元 tuple 不会形成独立的运行时形态。

## 方法与 Concept

结构体通过成员函数支持实例方法：

```c
struct User {
  Int id;

  Int value() {
    return self.id;
  }
}

User u = User { id: 42 };
Int b = u.value();
```

Concept 定义跨类型的共享接口：

```c
concept Hashable {
  UInt hashU() const;
}

impl Hashable for Int {
  UInt hashU() const {
    return UInt(self);
  }
}

Int h = 123.hashU();
```

参见[泛型与多态](/zh/language/generics)了解 concept 约束与类型边界。
