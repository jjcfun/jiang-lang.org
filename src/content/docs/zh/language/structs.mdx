## Struct

`struct` 是名义类型。即使两个结构体字段完全相同，它们也是不同类型。

```c
struct Point {
    Int x;
    Int y;
}

struct Offset {
    Int x, y;
}
```

同类型多字段声明只是语法糖，`Int x, y;` 等价于分别声明 `Int x;` 和 `Int y;`。

## 字段字面量初始化

没有自定义 `init` 的 `struct` 可以使用字段字面量初始化：

```c
Point point = Point { x: 0, y: 0 }
Point^ heap_point = new (Point { x: 100, y: 200 });
```

拥有所有权的指针不会在赋值、初始化或算术这类值上下文中自动解引用；但字段访问会自动穿过 `T^`，因此可以直接访问字段：

```c
print("x=%d y=%d", heap_point.x, heap_point.y);
```

## 字段可变性

字段能否被写入由字段类型自身决定，不由外层变量是否带 `!` 决定：

```c
struct User {
    Int id;
    Int! age;
    UInt8[]? nick_name;
}

User user = User { id: 123, age: 18 }
user.age = user.age + 1;
// user.id = 200; // 编译错误：字段自身不可变
```

可选字段可以默认初始化为 `null`。如果字段有默认值或是可选字段，构造时可以省略对应字段。

## `init`

`init` 是 `struct` 的构造器入口。它不声明返回类型，函数体内有正在初始化的 `self`：

```c
struct Point {
    Int x;
    Int y;

    init(Int x, Int y) {
        self.x = x;
        self.y = y;
        return ();
    }

    init(Int value) {
        self.x = value;
        self.y = value;
        return ();
    }
}

Point p1 = Point(1, 2);
Point p2 = Point(3);
Point^ p3 = new Point(4, 5);
```

规则：

- `init` 隐式拥有 `self`，不声明返回类型。
- 可定义多个 `init`，按参数数量和类型重载。
- 只要类型定义了 `init`，就不再使用 `Point { ... }` 字段字面量构造。
- `init` 可以写可见性修饰，例如 `public init(...)`。
- `init` 只能通过 `Type(...)` 或 `new Type(...)` 调用，不作为普通函数值暴露。

## `deinit`

`deinit` 是 `struct` 的析构入口。它由作用域退出、拥有所有权的指针析构和拥有所有权的字段析构触发；`ptr$.free()` 是低层释放操作，不作为普通析构入口使用。

```c
struct Buffer {
    UInt8[*] data;

    deinit() {
        self.data$.free();
        return ();
    }
}
```

如果 `struct` 有 `T^` 字段，离开生命周期时这些拥有所有权的字段会被自动释放。若结构体同时定义了自定义 `deinit`，会先执行自定义 `deinit`，再按字段声明逆序释放拥有所有权的字段。

## 方法

结构体支持 `static` 方法和实例方法：

```c
struct User {
    Int id;

    static Int zero() {
        return 0;
    }

    Int value() {
        return self.id;
    }
}

Int a = User.zero();
User u = User { id: 42 }
Int b = u.value();
```

实例方法体内有隐式 `self: Self&`。字段能否赋值仍只由字段类型自身是否带 `!` 决定，方法不需要额外的 `mutating` 标记。

## 记录类型

`record` 是偏数据记录的名义类型。它使用字段字面量初始化，字段支持默认值：

```c
record PointRecord {
    Int x;
    Int y = 2;
}

PointRecord p1 = PointRecord { x: 40 }
PointRecord p2 = PointRecord { x: 1, y: 2 }
```

常用规则：

- `record` 支持 `Type { field: value }`。
- `record` 字段支持默认值和同类型多名字声明。
- `record` 的定义体语法与 `struct` 共用，可以包含字段、方法、`init`、`deinit` 和关联类型实现。

## 局部声明

局部变量声明一次只绑定一个名字。需要多个局部变量时，写成多条声明：

```c
Int a = 1;
Int b = 2;
Int c = 3;
```
