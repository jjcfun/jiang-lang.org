Jiang 把控制流分成两类：改变执行路径的语句，以及能产生值的表达式。`if`、`switch` 和代码块可以在需要值的地方使用；`return`、`throw`、`break` 和 `continue` 是不会正常继续执行的语句。

## 代码块

```c
Int value = {
    Int base = 40;
    base + 2
}
```

代码块的值来自最后一个不带分号的尾表达式。没有尾表达式时，代码块的值是 `()`。

## `if`

`if` 条件后直接跟代码块，不需要强制写括号。作为语句使用时，`else` 可以省略：

```c
if value < 0 {
    print("negative");
}
```

作为表达式使用时，必须有 `else`，两个分支都要产生兼容的值：

```c
Int sign = if value < 0 {
    -1
} else {
    1
}
```

如果语句分支只执行语句，代码块没有尾表达式，结果就是 `()`：

```c
if value < 0 {
    print("negative");
} else {
    print("non-negative");
}
```

## `switch`

`switch` 用模式匹配分支。它可以作为表达式使用：

```c
enum Mode {
    read,
    write,
}

Mode mode = Mode.read;

Int code = switch mode {
    .read => 1,
    .write => 2,
}
```

`switch` 分支右侧可以是单个表达式，也可以是代码块：

```c
Int code = switch mode {
    .read => {
        print("read");
        1
    },
    .write => 2,
}
```

## `while` 循环

```c
Int! count = 0;
while count < 5 {
    count = count + 1;
}
```

`break` 结束当前循环，`continue` 进入下一轮：

```c
while count < 10 {
    count = count + 1;

    if count == 3 {
        continue;
    } else {
        print(count);
    }

    if count == 8 {
        break;
    }
}
```

## `for in`

`for 绑定 in 被遍历值` 用于遍历数组、切片和其他可遍历值。`in` 前面写绑定模式，`in` 后面写被遍历的值。当前内建支持数组和切片：

```c
Int[_] values = [10, 20, 30];

for item in values {
    print(item);
}
```

`for` 前面的绑定模式必须是不可失败绑定。需要解构元组时，在循环体内写独立解构语句：

```c
(Int, Int)[_] pairs = [(1, 2), (3, 4)];

for pair in pairs {
    (_ left, _ right) = pair;
    print(left + right);
}
```

Range 表达式写成 `start..end`，当前约定为左闭右开：

```c
for i in 0..10 {
    print(i);
}
```

## `guard`

`guard` 用于提前退出，并把成功匹配得到的绑定带到后续作用域。`else` 代码块必须非空，最后一条语句必须是 `return`、`break`、`continue` 或 `throw`。

```c
() print_value(Int? maybe) {
    guard maybe is some value else {
        return ();
    }

    print(value);
    return ();
}
```

## `defer`

`defer` 会在当前块退出时按后进先出顺序执行：

```c
defer handle$.free();

defer {
    log("closing");
    handle$.free();
}
```

`defer` 的语法是 `defer expr;` 或 `defer { ... }`。

## `try catch`

`try expr catch (...) => 备用值` 用于把 `T@E` 结果显式转换成成功值 `T`。备用值会在失败时使用：

```c
Int value = try parse(text) catch (err) => {
    log_error(err);
    0
}
```

`catch ()` 表示不需要错误值：

```c
Int value = try parse(text) catch () => 0;
```
