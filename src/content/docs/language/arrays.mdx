Array length is part of the type and must be known at compile time.

## Arrays

```c
Int[3] values = [1, 2, 3];

// Length mismatch is an error.
// Int[5] bad = [1, 2, 3];
```

Heap arrays use an owning pointer:

```c
Int[3]^ owner = new [1, 2, 3];
print(owner.length);
```

## Length Inference

Use `_` in the array length or infer the whole type:

```c
_ a = [1, 2, 3];
Int[_] b = [1, 2, 3];
Int[3] c = [1, 2, 3];
```

## Mutability

Element mutability and outer binding mutability are separate layers:

```c
Int![_] items = [1, 2, 3];
items[1] = 4;

Int[_]! slot = [1, 2, 3];
slot = [4, 5, 6];

Int![_]! both = [1, 2, 3];
both[1] = 4;
both = [4, 5, 6];
```

## Nested Arrays

Jiang does not need a separate multidimensional array form. Repeated array suffixes build nested arrays:

```c
Int[2][3] matrix = [[1, 2], [3, 4], [5, 6]];

matrix[0];    // [1, 2]
matrix[0][1]; // 2
```

Read `Int[2][3]` as `Int -> Int[2] -> Int[2][3]`.

```c
Int?[2][3] nullable_items = [[1, null], [3, 4], [5, 6]];
Int?[2]?[3] nullable_rows = [[1, null], null, [5, 6]];
```

## Slices

`T[]` names an unsized array pointee. Use `T[]&` for a borrowed runtime-length view over contiguous
storage. The borrowed slice does not own the storage.

```c
Int[_] values = [1, 2, 3];
Int[]& view = values[..];

print(view.length);
```

Sentinel slices use `T[:S]&`. A string literal naturally has type `UInt8[:0]&`; use `text.slice()`
or `text$.as(UInt8[]&)` when a non-literal sentinel slice must be viewed as a plain `UInt8[]&`:

```c
UInt8[:0]& text = "hello";
UInt8[]& bytes = text.slice();
```

Indexing a slice can produce an element reference:

```c
_ item_ref = view[1]$.ref();
```

`T[]&` is a borrowed view and does not own the underlying storage. `Vector<T>.slice()` returns such a
borrowed view. Use `Vector<T>.into_array()` when the vector should be consumed and its initialized
items should become an owned `T[]^`; the original vector is moved and cannot be used afterward.

## Many Raw Pointers

`T[*]` is a raw pointer form for contiguous elements without a length:

```c
UInt8[*] data;
Int length;

UInt8 first = data[0];
```

`T*` is a single-object raw pointer and is not used for free position movement. Use `T[*]` for
position-based contiguous-memory access, and use `T[]&` when Jiang code needs a borrowed
length-bearing view.
