## if-else

```c
if condition {
  // ...
} else if other_condition {
  // ...
} else {
  // ...
}
```

if expressions can return values (both branches must produce compatible types):

```c
Int x = if true { 1 } else { 0 };
```

## for-in

Ranges are exclusive on the right:

```c
for i in 0..10 {
  print("i = %d", i);  // prints 0 through 9
}
```

Iterate over elements:

```c
Int[] ints = [1, 2, 3];
for i in ints {
  print("i = %d", i);
}
```

Break with conditional:

```c
for i in 0..10 {
  if i > 3 break;
}
```

- The loop variable `i` is immutable by default
- There is no C-style `for` loop in Jiang

## while

```c
while condition {
  // ...
}
```

## break & continue

`break` exits a loop, `continue` skips to the next iteration, optionally with a label:

```c
outer: for i in 0..3 {
  for j in 0..3 {
    if i + j > 2 break outer;
  }
}
```

## goto

```c
readagain: {
  goto readagain;
}
```

## defer

`defer` executes code when the current scope exits. Multiple defers run in LIFO order:

```c
{
  defer print("this runs second");
  defer print("this runs first");
  print("normal execution");
}
// Output:
//   normal execution
//   this runs first
//   this runs second
